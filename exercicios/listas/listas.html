Trabalhando com listas

Listas Ordenadas

Uma lista ordenada é quando a sequência importa, a tag usada é OL e cada item da lista tem a tag LI.

Os tipos de indice são: 1 para numérico, A e a para alfabético, I e i para algarismo romano

  1. Acordar
  2. Tomar cafe

Listas não ordenadas

Uma lista não ordenada é quando a sequência não importa, a tag usada é UL e cada item da lista tem a tag LI.

Os tipos de indice são: disk para bola preenchida, circle bola não preenchida e square quadro.

  • Pão
  • Queijo

Misturando tipos de listas

É possivel misturar listas ordenadas(OL) com listas nao ordenadas (UL)
